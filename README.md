### 📖 About This Repository
Welcome to my official internship portfolio! This repository contains the projects and tasks I developed during my **Web Development Internship at InternPe**. 

The goal of this internship was to build responsive, user-friendly, and interactive web applications from scratch, utilizing modern front-end technologies and design principles like Glassmorphism and Neumorphism.

---

### 🗂️ Project Directory

#### 🧮 [Task 1: Simple Calculator](./Task_1_Simple_Calculator)
A functional and visually striking calculator featuring a 360-degree spherical UI design, neumorphic active states, and full arithmetic capabilities.
* **Live Demo:** [View Task 1 Here](https://jmchavda2006-glitch.github.io/Web-Development-Internship-Tasks-of-InternPe/Task%201%20Simple%20Calculator/)

#### 🛒 [Task 2: LimeCart E-Commerce Platform](./Task_2_E-Commerce)
A fully functional front-end e-commerce store featuring 25 dynamic products, advanced category filtering, real-time search, interactive shopping cart logic, and a premium glassmorphism UI.
* **Live Demo:** [View Task 2 Here](https://jmchavda2006-glitch.github.io/Web-Development-Internship-Tasks-of-InternPe/Task%202%20E-Commerce%20Website/)

---

### 🛠️ Tech Stack & Skills Demonstrated
Throughout these tasks, I utilized and strengthened the following technologies:
* **HTML5:** Semantic structuring and accessible layout design.
* **CSS3:** Advanced styling including CSS Grid/Flexbox, custom variables, keyframe animations, responsive media queries, and Glassmorphism effects.
* **Vanilla JavaScript:** DOM manipulation, array filtering/mapping, dynamic HTML injection, event listeners, and `localStorage` integration.

---

### 📥 How to Explore Locally
1. **Clone or Download** this entire repository to your local machine.
2. Navigate into either the `Task 1` or `Task 2` subfolders.
3. Open the respective `index.html` file in any modern web browser to view the projects.
