### 📖 Project Overview
This project is a fully functional, front-end **E-Commerce Store (LimeCart)** built during my Web Development Internship at **InternPe**. It features a modern, responsive glassmorphism UI styled with a custom lime and emerald green color palette. 

Key features engineered using pure **HTML, CSS, and JavaScript** include:
* **Interactive Shopping Cart:** Persistent cart logic with real-time add, quantity update, and subtotal calculation features.
* **Dynamic Filtering & Search:** A blurred pop-up modal for category filtering (Electronics, Fashion, Home Appliances, etc.), price range adjustments, and live text search.
* **Detailed Product Pages:** Clickable product cards that seamlessly transition into a dedicated view with image galleries and detailed descriptions.
* **Premium UX/UI:** Includes a smooth splash screen entry animation, dynamic UI rendering, and a custom developer profile modal.

### 📥 How to use this code
1. **Download** the project folder from this repository.
2. **Open** `index.html` in any web browser.
3. **Experiment** with the code by opening `script.js` to customize the mock product database, add your own images, or tweak the CSS theme variables!
